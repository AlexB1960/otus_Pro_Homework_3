# Homework N3

Проводится API-тестирование метода GET /pet/findByStatus и метода POST /pet/findByStatus (по свагеру https://petstore.swagger.io).
Используются спецификации RequestSpecification и ResponseSpecification, а также валидация
json-схем postpetschema.json и getpetschema.json. Необходимо использовать VPN.
